# praca_magisterska

W mojej pracy magisterskiej zajmowałem się problemem przycinania sztucznych sieci neuronowych. Praca miała charakter badawczy, a w przeprowadzonych eksperymentach ograniczyłem się do perceptronu wielowarstwowego (MLP). Badany był wpływ wielkości przycięcia sieci (usunięte neurony) w zależności od dopuszczalnej utraty jakości jej działania. W pracy porównywane były 4 algorytmy przycinania niestrukturalnego (usuwającego wagi między neuronami, a następnie neurony do których wchodziły same połączenia z zerowymi wagami) oraz 5 algorytmów przycinania strukturalnego (usuwającego neurony w całości). W wyniku przeprowadzonych badań stwierdziłem, że nie można wskazać jednoznacznie, który typ metod przycinania (strukturalne czy niestrukturalne) pozwala na większe przycięcie sieci; jednakże zastosowanie przycinania strukturalnego pozwalało na osiągnięcie minimalnie lepszych wyników dla zadania klasyfikacji.
